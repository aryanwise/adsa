# A.D.S.A (Autonomous Data Science Agent)

[![Live App](https://img.shields.io/badge/Live%20Demo-Streamlit-FF4B4B?style=for-the-badge&logo=streamlit)](https://autonomousdatascienceagent.streamlit.app/)

Welcome to the **Autonomous Data Science Agent (ADSA)**—an intelligent orchestration framework that completely automates the machine learning lifecycle. Provide a raw dataset, and ADSA will autonomously profile the data, engineer features, train and evaluate models, and write an executive summary of its findings, completely hands-free.

---

## How It Works

ADSA operates via three distinct, specialized AI agents executing in sequence:

- **Phase 1: The Architect (Planning)**
  Ingests the raw data, performs statistical profiling, and writes a strict Markdown blueprint/contract dictating how the pipeline should be built.
- **Phase 2: The Coder (Execution & Self-Healing)**
  Dynamically generates and executes Python scripts in a sandboxed environment to build the ML pipeline. If a script crashes, the Coder reads the `stderr` traceback and autonomously rewrites the code until it succeeds.
- **Phase 3: The Analyst (Reporting)**
  Compiles cross-validation metrics, holdout confusion matrices, and execution logs into a final, executive-grade Markdown report.

---

## The Streamlit Console

The entire operation is visualized and monitored via a sleek, unified Streamlit dashboard.

**Key Features Include:**

- **Headline Metrics:** Top-level vital stats tracking phase completion (Plan, Execute, Report), total steps executed, autonomous script repairs, the winning model, and the final holdout F1 score.
- **Executive Summary:** A clean render of the final, executive-grade Markdown report generated by the Analyst AI detailing project goals and methodology.
- **Model Leaderboard:** Visualizations of cross-validation metrics and holdout evaluation matrices to compare model performance.
- **Dataset Overview:** An LLM-synthesized narrative explaining the dataset's business dimensions, categorical groups, and statistical nuances.
- **Data Explorer:** Direct access to active dataset previews and raw statistical JSON profiles.
- **Pipeline Blueprint:** The literal Markdown contract the Architect AI wrote to guide the Coder AI's execution phase.
- **Engine Logs:** Expandable real-time engine logs showcasing standard output, standard error, generated Python scripts, and auto-repair attempts for full transparency.
